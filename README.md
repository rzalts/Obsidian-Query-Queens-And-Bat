## 📌 OBSIDIAN – Runway Production Management System

OBSIDIAN is a web-based platform designed to streamline the complex workflow of runway show production. It provides a centralized interface where coordinators and backstage teams can manage all aspects of a fashion show in real time, eliminating the need for fragmented tools or manual tracking.

The system is built as a browser-accessible application, allowing users to work from any device without installing additional software. OBSIDIAN handles highly interconnected data such as collections, looks, models, garments, fittings, alterations, and show schedules, making it easier to organize and update critical information efficiently.

### 🚀 Features
- **Centralized Dashboard** for managing all runway production data  
- **CRUD Operations**: Create, update, delete, and monitor records seamlessly  
- **Show Management**: Organize collections, looks, and runway sequencing  
- **Model & Garment Tracking**: Link models, items, and storage locations  
- **Fittings & Alterations**: Track preparation progress before shows  
- **Real-time Updates** for improved coordination across teams  

### 🎯 Target Users
OBSIDIAN is built for runway show coordinators and backstage production teams who handle the logistical complexity of fashion events. The platform helps users manage multiple moving parts simultaneously while ensuring accuracy, efficiency, and smooth execution of shows.

### 🎨 Design
The user interface and workflow were prototyped in Figma, showcasing key interactions such as authentication, show creation, and detailed data management across all system components.
